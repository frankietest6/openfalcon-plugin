OpenFalcon Plugin Configuration

Connects FPP to your self-hosted OpenFalcon server. See documentation for details.

Server Connection
e.g. http://192.168.1.230:3100 or https://lights.yourdomain.com
No trailing slash.
From your OpenFalcon admin dashboard.

Playlist Setup
The FPP playlist containing your viewer-controllable sequences.
Reads the selected playlist and pushes its sequences to OpenFalcon. Run this whenever you add/remove sequences from the playlist.

Behavior
onChange="SetPluginSetting('','interruptSchedule',encodeURIComponent(this.checked ? 'true' : 'false'));" /> If checked, viewer requests play immediately. Otherwise they queue after the current song.
How many seconds before end of current song to fetch the next request. Default: 3.
Extra seconds to wait after queueing. Default: 0.
How often to poll FPP's status API. Default: 1.

Logging
onChange="SetPluginSetting('','verboseLogging',encodeURIComponent(this.checked ? 'true' : 'false'));" /> Log every API call (noisy). Log file: /home/fpp/media/logs/openfalcon-listener.log
After changing settings, restart the listener:
Use the command "OpenFalcon — Restart Listener" from the FPP command scheduler, or restart FPP.
